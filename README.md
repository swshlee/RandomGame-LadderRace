# 랜덤 사다리 경주

엑셀 또는 CSV 파일의 제출 순서대로 사람을 Column에 배치하고, 각 사람이 제출한 최대 3개의 숫자로 오른쪽 Column과 연결되는 Row를 만드는 사다리 게임입니다.

## 실행

GitHub Pages:

```text
https://swshlee.github.io/RandomGame-LadderRace/
https://swshlee.github.io/RandomGame-LadderRace/?demo=1
```

로컬에서는 `index.html`을 브라우저로 열거나 정적 서버로 실행하면 됩니다.

## 입력 파일

첫 번째 시트 또는 CSV 첫 행에 아래 컬럼이 필요합니다.

- `이름` 또는 `참여자명`
- `숫자`, `숫자1`, `번호` 또는 `번호1`
- 선택 컬럼: `숫자2`, `숫자3` 또는 `번호2`, `번호3`

규칙:

- 엑셀 행 순서가 제출 순서입니다.
- 참여자 수만큼 왼쪽부터 오른쪽으로 Column이 생성됩니다.
- Column 높이는 `0~1000`입니다.
- 숫자는 한 사람당 최대 3개까지 입력할 수 있고, `1~999` 사이의 정수만 유효합니다.
- 숫자 하나의 셀에 `120, 300, 700`처럼 구분해서 적어도 최대 3개까지 읽습니다.
- 마지막 참여자는 오른쪽 Column이 없으므로 연결 Row를 만들지 않습니다.

## 게임 진행

파일을 업로드하면 참가자 데이터가 준비되고 `게임시작` 버튼이 활성화됩니다.
`게임시작`을 누르면 Column 번호, Column, 연결 Row가 순서대로 생성됩니다.
사다리가 완성되면 `GO` 버튼이 나타납니다.
샘플 데이터를 바로 확인하려면 `?demo=1` 주소로 접속합니다.
`GO`를 누르면 참여자 수 안에서 랜덤 시작 Column이 뽑히고, 하트가 사다리를 타고 내려갑니다.
하트가 마지막으로 도착한 Column의 사람이 우승자입니다.
